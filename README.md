# Interactive-Maths-Tools

**Interactive math teaching tools designed for classroom projection and whiteboard use.**

---

## What Is This?

A collection of React-based teaching tools that run instantly in Claude.ai with no installation required. Each tool generates unlimited practice questions across three difficulty levels.

## Goals

- **Classroom-Ready:** Large text optimized for projection, touch-friendly buttons for interactive whiteboards
- **Pedagogically Sound:** Three modes supporting "I Do, We Do, You Do" teaching (Whiteboard, Single Question, Worksheet)
- **Zero Friction:** Copy code → paste into Claude → tool runs immediately
- **Differentiation Built-In:** Three difficulty levels in every tool
- **Teacher Control:** Generate questions on-demand, show/hide answers, print worksheets instantly

## How It Works

All tools follow a consistent [Core System Specification](SPECIFICATION (TSX) v1.1) ensuring uniform behavior, visual design, and mathematical accuracy across topics.

**Typical Use:**
1. Open Claude.ai
2. Copy a tool from this repository
3. Paste into Claude
4. Project on classroom screen
5. Teach
